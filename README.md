# ToyShare 🧸

> Share joy, one toy at a time.

ToyShare is a Flutter app that lets neighbors share, lend, and donate toys to children in their community.

---

## Features

- **Auth** — Sign up, login, password reset
- **Browse Toys** — Search, filter by type (exchange/donate), tap owner to see public profile
- **List a Toy** — Add photo, set condition, category, age group, fee
- **Exchange Duration** — Set by date+time or by quick duration (5 min, 2 hrs, etc.)
- **QR Receipt** — Every exchange generates a QR code receipt with toy name, owner, return date
- **QR Scanner** — Scan another user's QR to verify the exchange
- **Incident Report** — Report lost, stolen, or damaged toy from the receipt screen
- **Chat** — In-app messaging per exchange with unread badges
- **Notifications** — Return reminders (same day, tomorrow, 3 days, overdue, 30-min warning for hourly loans)
- **Ratings & Reviews** — Leave feedback after exchange; public profiles show ratings
- **Dark Mode** — Full dark mode support across every screen
- **Web Support** — Runs on web with mobile layout

## Setup

### 1. Clone & install

```bash
git clone https://github.com/yourusername/toyshare.git
cd toyshare
flutter pub get
```

### 2. Firebase

```bash
npm install -g firebase-tools
dart pub global activate flutterfire_cli
flutterfire configure
```

This generates `lib/firebase_options.dart`.

### 3. Firestore Rules

Paste into Firebase Console → Firestore → Rules:

```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    function isAuth() { return request.auth != null; }
    match /toys/{id}         { allow read, write: if isAuth(); }
    match /requests/{id}     { allow read, write: if isAuth(); }
    match /exchanges/{id}    { allow read, write: if isAuth(); }
    match /messages/{id}     { allow read, write: if isAuth(); }
    match /notifications/{id}{ allow read, write: if isAuth(); }
    match /feedback/{id}     { allow read, write: if isAuth(); }
    match /blacklist/{id}    { allow read, write: if isAuth(); }
    match /disputes/{id}     { allow read, write: if isAuth(); }
    match /incidents/{id}    { allow read, write: if isAuth(); }
  }
}
```

### 4. Firestore Indexes

Create these composite indexes in Firebase Console → Firestore → Indexes:

| Collection    | Field 1           | Field 2          |
|---------------|-------------------|------------------|
| toys          | ownerId ASC       | createdAt DESC   |
| toys          | isAvailable ASC   | createdAt DESC   |
| exchanges     | ownerId ASC       | createdAt DESC   |
| exchanges     | holderId ASC      | createdAt DESC   |
| requests      | ownerId ASC       | status ASC       |
| notifications | userId ASC        | createdAt DESC   |
| messages      | exchangeId ASC    | createdAt ASC    |
| messages      | receiverId ASC    | isRead ASC       |
| feedback      | toUserId ASC      | createdAt DESC   |
| incidents     | reporterId ASC    | createdAt DESC   |

### 5. Run

```bash
flutter run
```
